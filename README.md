# FAKE-VS-REAL-IMAGES


This project is an AI-based image classification system that detects whether an image is **Real** or **Fake** using Deep Learning.

## 🚀 Features
- Upload an image
- Detect Real or Fake images
- User-friendly interface
- Fast image prediction

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit

## 📂 Project Structure
```
FAKE-VS-REAL-IMAGES/
│── dataset/
│── model/
│── app.py
│── requirements.txt
│── README.md
```

## ▶️ How to Run

1. Clone the repository
```bash
git clone https://github.com/Aswaniprakash1/FAKE-VS-REAL-IMAGES.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
streamlit run app.py
```

## 📸 Output
The application predicts whether the uploaded image is **Real** or **Fake**.
